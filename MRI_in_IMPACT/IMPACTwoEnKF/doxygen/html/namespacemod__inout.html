Detailed Description

module containing subroutines for reading and writing data to hdf5 files. It uses modules mod_dims, mod_vars, mod_exchange, mod_lib and HDF5.

Todo:
: information on the immersed boundary could be written to files with routines that are added here. If not, there should be routines in the structural solver.
