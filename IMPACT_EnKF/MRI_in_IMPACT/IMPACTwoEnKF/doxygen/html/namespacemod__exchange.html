Detailed Description

module holding subroutines for value exchange between blocks (ghost cells). It uses modules mod_dims, mod_vars

Todo:
: there is two practically identical subroutines exchange and exchange2, with the latter containing variable interval limits. Could one be obsolete? If so, perhaps every call to the obsolete routine could be updated in the entire code and the obsolete routine could be removed.
